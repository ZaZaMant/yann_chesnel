Création d’une Base De Donnée (D)

Projet réalisé en binôme, il nous fallait réaliser une Base de Données pour gérer un établissement ( ce qui comprend les élèves, les enseignants, leurs matière, les heures de cours…). En premier lieu on aura réaliser un schéma UML sur Visual Paradigme en analysant les besoins et les contraintes. Et enfin il ne nous restait plus qu’à créer la Base de Données, qui fût finalisée en SQL sur SQL Workbench.
