Création d’un site vitrine streaming

Site créé dans un groupe de 4, avec pour série présenté : Stranger Things. Un ordre d’instruction nous a été donné, nous avons donc du suivre les étapes suivantes : -création d’une arborescence avec le logiciel web Gloomaps. -La définition de chaque pages (donc leur structure et l’organisation) -Création de la charte graphique (couleurs, typographie, logos…) -Création d’une maquette a l’aide du logiciel web Figma. (maquette interactive) -Création du site. Ce qui nous a permis de découvrir de nombreux outils et de mieux organiser notre travail. Nous avons utilisé du CSS, du HTML, et du Javascript pour créer le site
